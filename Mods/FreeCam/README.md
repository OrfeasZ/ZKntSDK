# FreeCam Mod

## Controls

### Keyboard

| Keys | Action |
| ---- | ------ |
| K | Toggle freecam |
| F3 | Freeze camera and enable player input |
| Mouse | Rotate camera |
| W/S | Move camera forward/backward |
| A/D | Move camera left/right |
| Q/E | Move camera down/up |
| Arrow Keys | Move camera |
| Ctrl + W/S | Change FOV |
| Ctrl + A/D | Roll camera |
| Ctrl + X | Reset roll |
| Ctrl + Z | Reset FOV |
| Alt + W/S | Change translation speed |
| Alt + A/D | Change rotation speed |
| Alt + Z | Reset camera speed |
| F + W/A/S/D | Rotate camera in fixed increments (15° by default) |
| Space + W/A/S/D | Move camera in world space |
| Space + Q/E | Move camera vertically in world space |
| Shift | Increase speed while holding |
| Ctrl + F6 | Teleport player |
| F8 | Pause/Resume game |

### Controller

| Keys | Action |
| ---- | ------ |
| Right Analog Stick | Rotate camera |
| Left Analog Stick | Move camera |
| Right Bumper | Move camera vertically |
| Y + Left Analog Stick | Change FOV |
| A + Left Analog Stick | Roll camera |
| B + Left Analog Stick | Change camera speed |
| Left Analog Stick Press | Reset roll, FOV, and speed |
| Left Trigger + Left Analog Stick | Move camera in world space |
| Left Trigger + Right Analog Stick | Move camera vertically in world space |
| Right Trigger | Increase speed while holding |
| Left Bumper | Freeze camera and enable player input |

## Editor Style Controls

### Keyboard

| Keys | Action |
| ---- | ------ |
| K | Toggle freecam |
| F3 | Freeze camera and enable player input |
| MMB | Drag camera |
| Scroll Wheel | Zoom camera |
| RMB | Activate rotate |
| Alt + MMB or RMB | Orbit camera |
| Z + Alt + MMB or RMB | Orbit camera around selected entity |
| Z | Zoom to selected entity (press twice to focus the gizmo) |
| Alt + Scroll Wheel | Precision zoom |
| Shift | Speed modifier |
| RMB + Scroll Wheel | Adjust speed |
| F8 | Pause/Resume game |
